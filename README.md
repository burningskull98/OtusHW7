Django Blog Tutorial Project
Этот проект представляет собой адаптацию официального Django tutorial под веб-сервис для блогинга.
Он включает базовую функциональность создания и просмотра постов, покрыт тестами и сделан production-ready.


Описание
Проект основан на Django и реализует простой блог, где пользователи могут создавать посты (требуется аутентификация). 
Адаптация включает:
Модель Post вместо Question/Choice.
Представления для индекса, деталей поста и создания постов.
Шаблоны для отображения.
Админку для управления.
Дополнительно добавлены тесты и настройки для production: контейнеризация, переменные окружения,  использование PostgreSQL и т.д.
